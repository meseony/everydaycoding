웹사이트 완성

웹페이지 하나하나를 잇는 링크는 책을 엮는 실같은 역할, 책은 웹사이트라고 생각한다.

서버와 클라이언트

인터넷이 동작하기 위해서 최소 2대의 컴퓨터가 필요하다.
Web Browser와 Web Server 프로그램이 깔린 두 컴퓨터는 인터넷으로 연결되어 있다.

Web Server 컴퓨터는 http://info.cern.ch라는 주소를 가지고 있다고 하자.
그리고 Web Server의 하드디스크 디렉토리 안에 index.html이라는 파일이 저장되어 있다.
이때, Web Browser의 주소창에 http://info.cer.ch/index/html을 입력하면
인터넷을 통해 전기신호를 http://info.cern.ch에 해당되는 컴퓨터에 보낸다.
그 신호 안에는 "저는 'index.html'이라는 파일을 원합니다."라는 뜻이 담겨있다.
그러면 http://info.cern.ch에 해당되는 컴퓨터에 설치된 Web Server라는 프로그램이
하드디스크에서 index.html이라는 파일을 찾아 Web Broswer이 설치된 컴퓨터에 전기 신호로 보내준다.
이 신호에는 index.html이라는 파일 안에 저장되어 있던 코드가 담겨있다.
Web Browser의 컴퓨터는 index.html이라는 파일의 내용(코드)이 도착하는 것이다.
이제 Web Browser가 이 코드를 읽어 해석한 후, 화면에 표시한 것이 우리가 알고 있는 웹, 웹사이트가 동작하는 것이다.

Web Browser 컴퓨터는 정보를 요청하는 클라이언트(client),
Web Server 컴퓨터는 정보를 응답하는 서버(server)
우리가 배우고 있는 웹브라우저는 웹 클라이언트라고도 부를 수 있다.

웹서버에 대해서 웹서버를 쓸 수 있게 된다는 것은, 내가 만든 컨텐츠를 전세계 인터넷 사용자 누구에게나 사용할 수 있도록 한다는 것이다.
두 가지 방법이 있다.
1. 자신의 컴퓨터에 직접 웹서버를 설치하는 것. 어렵지만 그 과정에서 많은 것을 알게 된다.
2. 웹서버를 제공해주는 업체를 이용한다. 이를 웹호스팅 업체라고 한다. 쉽지만 많은 것이 감춰져있다.
'생활코딩'에서는 웹호스팅으로 쉽게 목표를 달성한 후에 웹서버를 직접 설치하는 방법을 공부하는 걸 추천한다.
